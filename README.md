# Noshi – A Romantic Apology Website

A fully static romantic website built for GitHub Pages or Netlify.

## Run locally
Open `index.html` in a browser, or use a simple local server.

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and folders from this project.
3. Go to Settings → Pages.
4. Choose Deploy from branch, select `main` and `/root`.
5. Save.

## Deploy to Netlify
Drag and drop the whole project folder or ZIP into Netlify Drop.
